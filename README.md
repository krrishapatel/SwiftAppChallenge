# Hospital Adventure

A SwiftUI app to help children prepare for hospital visits. Created for the Swift Student Challenge.

## Overview

Hospital Adventure guides children through "The Healing Grove" — a comforting reimagining of the hospital experience. With Pip the Pinecone as their bravery buddy, children explore each step (arrival, check-in, changing room, IV, anesthesia, recovery) in a safe, interactive way before their actual visit.

## How to Run

1. **In Xcode:** Open `HospitalAdventure.swiftpm` (File > Open, select the .swiftpm folder)
2. **In Swift Playgrounds:** Open the project and run the app
3. Select the HospitalAdventure scheme and run on a simulator or device

## Requirements

- Swift 5.9+
- iOS 17+ / macOS 14+
- Xcode 15+ or Swift Playgrounds 4.6+

## Features

- **8 interactive scenes** — Welcome, Arrival Garden, Check-In Nest, Changing Room, Courage Vines, Rest Chamber, Cozy Nest, Bravery Ceremony
- **Pip the Pinecone** — A lovable companion who goes through each step with the child
- **Personalization** — Child enters their name; chooses what comfort item to bring
- **Bravery Badge** — Completion certificate to print or draw
- **Offline** — No network required; all content included locally
